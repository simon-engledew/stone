If you were previously using the Haml syntax:

<div class="code"><pre><code>%p= 'string ', method, ' string'</code></pre></div>

you will need to change it to:

<div class="code"><pre><code>%p= ['string ', method, ' string']</code></pre></div>

to avoid compilation errors in production mode.
